React and Single Page Applications

React is a leading front-end library used by companies like Facebook, Netflix, and more. Because React is a library (and not a framework), it provides enormous flexibility when building rich user interfaces and has developed a passionate community with excellent web developer and software development support.

Using React, learn to build single page applications — progressive web applications that feel like a native app. This modern software development approach is used by companies like Twitter and Facebook, and you'll learn how to deploy it in your own web development projects.
